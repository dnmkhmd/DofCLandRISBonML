# Car Leasing & Rental System (ML-Powered)

## Overview
This is a web-based platform for car leasing and rental, enhanced with Machine Learning capabilities.

## Tech Stack
- **Frontend**: Next.js (React), Vanilla CSS
- **Backend**: Python (FastAPI)
- **Database**: TBD
- **ML**: Scikit-Learn, Pandas

## Quick Start

1. **Install dependencies**:
   ```bash
   chmod +x install.sh start.sh
   ./install.sh
   ```

2. **Run the application**:
   ```bash
   ./start.sh
   ```

## Project Structure
- `frontend/`: Next.js application
- `backend/`: FastAPI service
